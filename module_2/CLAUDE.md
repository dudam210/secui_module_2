## Architecture

This is a vanilla JavaScript web application with the following architecture:

- **HTML**: Semantic HTML structure in `index.html`
- **CSS**: Styles organized in `css/` directory
- **JavaScript**: Client-side logic in `js/` directory
- **Assets**: Static resources in `assets/` directory

The application follows a simple frontend-only architecture without a build step, making it easy to develop and deploy.
